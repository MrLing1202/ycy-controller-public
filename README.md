# ⚡ YCY Controller

AI驱动的BLE设备智能控制引擎。

---

## 快速开始

```bash
cp .env.example .env
# 编辑 .env 填入BLE协议和AI模型配置

pip install -r requirements.txt
python run.py
```

浏览器打开 http://localhost:8080

---

## 功能总览

### 🤖 AI人格控制
- 3种AI人格 + 随机模式，每种有独特行为模式
- AI实时决策，每步输出ABC三通道速度
- AI自动在全停/单通道/双通道/三通道间切换
- 不能连续5步不给0值，合理安排休息
- 功率耦合：吮吸开启时自动降转速

### 💬 AI对话
- 文字对话，AI回复1-2句并调整设备参数
- 语音消息，长按录音发送
- 停止时AI会嘲讽、戏谑玩家

### 👍👎 反向反馈
- 点👍 → AI认为你在享受 → 收回强度(edge/tease)
- 点👎 → AI认为你在抗拒 → 加大强度

### 🎤 声音寸止
- 实时麦克风监听，分析音量/音调/变化速率
- 声音越大越尖持续越久 = 越接近 → 自动停止设备

### 🎲 随机模式
- 三通道随机跳变，休息率可调，间隔可调
- 开/关切换，同时控制设备和AI

### ⏸ 停止按钮
- 设置0秒：永久停止（再按恢复）
- 设置N秒：暂停N秒后自动恢复
- 停止时通知AI，AI会嘲讽玩家

### 🎚️ 手动控制
- 三通道滑条，拖动实时发送
- 非线性速度映射，低端更细腻

### 🎭 自动模式
- 6种预设：🎲随机 🌊波浪 💓脉冲 📈渐进 🔀组合 📜自定义脚本
- 速度变化通过事件推送到手机蓝牙

### ⏺ 录制回放
- 录下手动操作，回放或导出为JSON脚本

### 📱 蓝牙直连
- Web Bluetooth，手机浏览器直接连设备
- 不依赖服务器蓝牙

### 🛡️ 安全
- 吮吸通道自动放气
- 断连即停
- AI最大运行时长限制
- 声音寸止强制中断
